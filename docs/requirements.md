# Personal Website - Requirements

## Technology Stack
- **Framework**: Next.js (App Router)
- **Hosting**: Vercel
- **UI Library**: React 18+
- **Language**: TypeScript
- **Styling**: CSS Modules
- **Content**: MDX (Markdown + React components)

## Core Features

### 1. Home/About Section
- Landing page with introduction
- Professional bio
- Current role and company
- Education background
- Call-to-action (CTA) to projects/blog

### 2. Projects Section
- Grid/list of personal and professional projects
- Each project should include:
  - Title
  - Description
  - Technologies used
  - Link to live demo (if available)
  - Link to GitHub repo (if available)
  - Screenshot/image (optional)

### 3. Blog
- MDX-based blog posts stored in the repository
- Blog index page listing all posts
- Individual blog post pages with full content
- Post metadata: title, date, description, tags
- Tag/category filtering
- Search functionality
- Responsive design for reading

### 4. Contact Section
- Social media links (GitHub, LinkedIn, Twitter/X, etc.)
- Email address display
- Contact form with fields:
  - Name
  - Email
  - Message
  - Form submission handling (via API route or service)

## Design Requirements
- **Style**: Professional, minimal, technical aesthetic
- **Typography**: Clean, readable fonts
- **Colors**: Neutral palette with subtle accent colors
- **Layout**: Responsive (mobile-first approach)
- **Performance**: Fast loading, optimized images
- **Accessibility**: Semantic HTML, ARIA labels, keyboard navigation

## SEO & Meta
- Proper meta tags for each page
- Open Graph tags for social sharing
- Sitemap generation
- robots.txt

## Nice-to-Have Features (Future Enhancements)
- Dark mode toggle
- Reading progress indicator for blog posts
- Related posts suggestions
- Newsletter subscription
- RSS feed
- View counter for blog posts
- Code syntax highlighting in blog posts
